Климатические параметры отопительного периода
Градусо-сутки отопительного периода (ГСОП) определяется по формуле 5.2 СП 50.13330.2012
ГСОП = (tв - tот)·zот
где tв = {{norm.t_int}} °С - температура внутреннего воздуха помещений здания (определяется по ГОСТ 30494-96);
tот = {{norm.t_ot}} °С – средняя температура наружного воздуха отопительного периода для г.{{norm.citi}}, для периода со средней суточной температурой наружного воздуха не более {% if norm.typ == "Детское учреждение" %} 10 °С, определяется по табл. 1 СП 131.13330.2012; {% else %} 8 °С, определяется по табл. 1 СП 131.13330.2012; {% endif %}
zот = {{norm.z_ot}} °С – продолжительность отопительного периода для г.{{norm.citi}}, для периода со средней суточной температурой наружного воздуха не более {% if norm.typ == "Детское учреждение" %} 10 °С, определяется по табл. 1 СП 131.13330.2012; {% else %} 8 °С, определяется по табл. 1 СП 131.13330.2012; {% endif %}
ГСОП = ({{norm.t_int}} - ({{norm.t_ot}}))·{{norm.z_ot}} = {{norm.gsop}} ºС·сут

В соответствии с пунктом 5.2 допускается снижать нормируемое сопротивление теплопередаче чердачного перекрытия, если температура воздуха чердака отличается от температуры наружного воздуха. n = (tв* - tн*) / (tв - tн) = ({{norm.t_int}} - {{attic["elem"].t_ext}}) / ({{norm.t_int}} - ({{norm.t_ot}})) = {{attic["elem"].n_coef}}

tв = {{norm.t_int}} °С - нормативная температура внутреннего воздуха;
